# Sobre o sistema:
## Quem usará o sistema?
O usuários do sistema serão empresas ou pessoas geralmente usam os dados em obras, áreas de meterologia, estalagem, ou qualquer outro setor que precise de um cuidado, o maior índice é em obras.

## Onde?
Os usuários que usarão do sistema poderão o acessar pela web ou no celular, podendo acessar de casa, na rua ou no trabalho.

## Por que?
O sistema mostrará dados que poderão ser usados para planejamento de obra ou continuidade de uma já existente, podendo usar o clima ao seu favor.

## Quando?
Geralmente antes da obra ou da necessidade, as empresas buscam e junto dos dados fazem o planejamento. Algumas empresas recorrem durante e desde que a pessoa continue pagando a plataforma pode ser usada até mesmo depois de pronta para monitoramento.

## O que?
O sistema junta um mapa interativo junto de um gráfico que mostra diferentes dados, esses dados podem ser mostrado a partir de um click único ou usando polígonos para selecionar pontos no mapa.

## Como?
O sistema conta com uma assinatura mensal, o usuário faz um cadastro e depois consegue fazer login, o sistema fica aberto para acesso na web e poderá ser acessado quando o usuário quiser.  

## Quais dados?
Múltiplos dados são absorvidos pelo sistema para gerar suas previsões, dados como:
- Dados de Radares
- Dados de raios
- Dados de estações
- Localização da obra em questão
